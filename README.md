## Installation
Install dependencies as specified in requirements.txt
e.g.
    sudo apt-get install python-pip
    sudo pip install -r requirements.txt

Make sure that the client is executed on startup:
    edit /etc/rc.local
    add:
        /<full path to>/client.py
